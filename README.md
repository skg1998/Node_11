# Node_11

> Repository to learn and create Node Project, Create a 11 node project with different specification to learn more much concept of node

## Usage

Clone this repository

## Install Dependencies

```
# project_folder/
npm install
```

## Run App

```
# Run server
npm run dev
```

## Access

```
# Access the Server:
 http://localhost:5000/
```
